### git常用命令

git init   初始化仓库

git clone ssh/http   从远端仓库拉取代码

git add .    将本地的修改全部提交到暂存区

git push  远端仓库名    将本地仓库提交到远端仓库

git status  查看当前分支状态

git checkout 还原某个文件

git reset   还原某次版本提交
